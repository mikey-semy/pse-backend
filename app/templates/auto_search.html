Инструкция по использованию:

  1. Скопируйте код закладки: Прямая ссылка
  2. Создайте новую закладку "Новой вкладки".
  3. Нажмите на закладку правой кнопкой мыши.
  4. В контекстном меню выберите "Изменить".
  5. Вместо URL вставьте скопированный код.
  6. Название закладки - любое или полностью удалите.
  7. Войдя в предсменный экзаменатор на любой странице нажмите на закладку
  8. Далее начинайти прохождение теста
  9. Для поиска ответа нажмите S

Информация по управлению:

  • S - Поиск правильного ответа из база данных
  • A - Добавление нового вопроса с ответами в базу данных (постарайтесь ответить правильно)
  • D - Обновление вопроса с ответами в базе данных (для исправления ошибочного ответа)

Проблемы:

  • Поиск вопроса по тексту, а не по уникальному номеру id
  • Одинаковые вопросы придется все время переписывать, так как по ним будут неправильные ответы
  • Добавить одинаковый вопрос невозможно
  • Сохранение текстовых вопросов невозможно
